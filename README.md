# Python Online Compiler

A full-stack Python Online Compiler built using React, Node.js, Express, and Flask.

This project allows users to write and execute Python code directly in the browser with real-time output.

---

# Features

* Python code execution
* Monaco Editor integration
* Real-time output display
* Full-stack architecture
* REST API communication
* Dark theme UI
* Error handling

---

# Tech Stack

## Frontend

* React.js
* Monaco Editor
* Axios
* CSS

## Backend

* Node.js
* Express.js

## Python Runner

* Flask
* Flask-CORS
* subprocess module

# Installation

## 1. Clone Repository

```bash
git clone https://github.com/niranjanswain067/python-online-compiler.git
```

---

## 2. Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## 3. Backend Setup

```bash
cd server
npm install
node server.js
```

---

## 4. Python Runner Setup

```bash
cd python_runner
pip install flask flask-cors
python app.py
```

---

# Usage

1. Write Python code in the editor.
2. Click "Run Code".
3. View output in the output panel.

---
